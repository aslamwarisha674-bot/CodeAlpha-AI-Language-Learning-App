# 🌍 CodeAlpha - Language Learning App

An interactive Language Learning Android Application developed as part of the CodeAlpha Internship Program using Google AI Studio, Kotlin, and Jetpack Compose.

This application is designed to make language learning simple, engaging, and interactive by helping users improve their vocabulary, grammar, pronunciation, and overall communication skills through quizzes, flashcards, and AI-powered learning features.

---

## 📱 Features

* 📚 Learn new words, phrases, and sentences
* 🗂️ Vocabulary and Grammar categories
* 📝 Interactive quizzes and practice tests
* 🔊 Pronunciation support
* 🎯 Daily learning activities
* 🤖 AI-assisted language learning experience
* 📱 Modern and responsive user interface
* ⚡ Smooth navigation and user-friendly design

---

## 🛠️ Tech Stack

* Kotlin
* Jetpack Compose
* Material Design 3
* Google AI Studio
* Android Studio

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aslamwarisha674/CodeAlpha-Language-Learning-App.git
```

### 2. Open the project

Open the project in Android Studio.

### 3. Sync Gradle

Allow Gradle to download and sync all required dependencies.

### 4. Run the application

Run the app on an Android Emulator or a physical Android device.
